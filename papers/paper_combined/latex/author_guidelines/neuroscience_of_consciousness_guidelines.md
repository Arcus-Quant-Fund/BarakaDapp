## Submission Checklist

### Format
- [ ] Editable file format — **NOT PDF** at initial submission (Word or LaTeX source)
- [ ] First page: title, all author names, all affiliations (department, institution, city, country)
- [ ] One corresponding author with mailing address + email
- [ ] Figures numbered (Figure 1, Figure 2…), referenced in text, with legends
- [ ] Figures inline is OK at initial submission; high-res separate files needed on revision
- [ ] Common image formats: pdf, eps, gif, tif, jpg

### Required elements
- [ ] ORCID iD for submitting author (register free at orcid.org if needed)
- [ ] CRediT contributor roles for each author
- [ ] Data Availability Statement (can say: "Code and data available at [OSF/GitHub link]")
- [ ] Conflict of interest disclosure
- [ ] Funding statement (or state "no funding")
- [ ] **AI use disclosure** — Claude was used in writing/editing; must state in Methods or Acknowledgements
- [ ] 5 suggested reviewers with contact details (consider diversity: career stage, gender, geography)
- [ ] Cover letter disclosing any preprints or related papers

### Word count check for v7.tex
Run: `texcount v7.tex` — target ≤9,000 words body text
